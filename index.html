Understanding Receiver Operating Characteristic (ROC) Curves

ROC Curves are thrown around a lot, and the overall thing to look for is that you want the curve to resemble a square and not a triangle.

FP = TP = 0

fprev = -10
i = 1

while (i < len(data)):
  if (data[i].score != fprev):
	R.append([TP/N, FP/N])
	fprev = data[i].score

  if (data[i].score):
	TP += 1
  else:
	FP += 1

R.append([TP/N, FP/N])
